# IPL Data Analysis

This repository contains analysis done on four datasets take from <a href ="https://www.kaggle.com/ramjidoolla/ipl-data-set">Kaggle</a><br>
  i)  matches.csv <br>
  ii) most_runs_average_strikerate.csv <br>
 iii) deliveries.csv <br>
  iv) Players.csv <br>

All datasets can be found in the *data* folder.
Visualizations were created using PowerBI.

#### Insights
1) Most Winning Team did fielding first.
